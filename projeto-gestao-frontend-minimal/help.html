Ajuda

Esta UI é estática (HTML+CSS). Conecte ao backend Java (Spring Boot/Jakarta) mapeando rotas e preenchendo os placeholders data-* e IDs.

Rotas sugeridas

  • GET /login, POST /login
  • GET /projects, POST /projects, GET /projects/:id
  • POST /tasks, PUT /tasks/:id, DELETE /tasks/:id

Acessibilidade

Campos com label, contraste adequado, navegação por teclado.
